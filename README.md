## CONTRIBUTING
If you have your own **Home Server**, or feel like, my Home Server architecture could be improvised-would love to see that. Make a new branch in the format:- 
1. `enh_main_<unique identifier>` → For enhancements in the current `main` branch *(well, if making, don't make the branch name something that I cannot keep)*.
2. `<architecture explanation>` → A completely new branch for a whole new architecture; be it multi-device clustering, or so.

When making the branch, do edit `README.md` which has a similar sections like `Introduction`, `Specifications`, `Architecture (dig optional)`, `Instructions`, `Branch Maintainer's Footer`
